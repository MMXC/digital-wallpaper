🔐 Slack 凭证配置

配置步骤：

4. 添加 Bot Token Scopes: chat:write, channels:history
5. 安装应用到工作区

1. 打开 Wallpaper Engine 编辑器
2. 选择 "创建空白项目"
3. 将项目目录下的 project.json 复制到壁纸项目目录
4. 或直接添加 web effect，URL 填: http://localhost:18791
